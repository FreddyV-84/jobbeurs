Onze Opleiding

Onze opleiding begon met een basis in programmatielogica gevolgd door grondige lessen over Javascript, HTML en CSS.

In Javascript implementeerden we AJAX waarbij we zowel XML als JSON bestanden leerden ophalen en verwerken in de DOM. De CSS-layouts Floats, flexbox en grid's zijn ons niet meer vreemd en semantische html5 tags maakten alles betekenisvoller. Met behulp van CSS Preprocessor SASS werd ons leven nog een stuk gemakkelijker. Ook JQuery, vrij vertaald door onze docent: 'voor mensen die geen javascript kennen', gaf ons een verhelderendere blik op de evolutie van het web met zijn browser compatibilteitsproblemen.

Ook Database Management Systemen zoals MariaDB en hun CRUD operaties zijn de revue gepasseerd. Alsmede het normaliseren van data en het gebruiken van SQL.

Tot slot zagen we het Javascript framework Angular4 en JS Framework (Angular4, Bootstrap) Content Management System (Drupal) SQL Node/npm Beheersysteem (Git/GitHub) Met behulp van node en het beheersysteem Github
